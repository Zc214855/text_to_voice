# FairyVoice 使用文档

## 启动

1. 确认已安装 Node.js。
2. 双击 `start.bat`。
3. 浏览器打开 `http://localhost:5173`。

## 配置

应用读取 `web/public/config.json`。

新版控制台 API Key 鉴权：

```json
{
  "apiKey": "your-api-key",
  "resourceId": "seed-tts-2.0",
  "resourceIds": [
    "seed-tts-2.0",
    "seed-tts-1.0",
    "seed-icl-2.0"
  ]
}
```

兼容旧版 AppId / Access Token 鉴权：

```json
{
  "appId": "your-app-id",
  "accessToken": "your-access-token",
  "resourceId": "seed-tts-2.0"
}
```

兼容旧字段：`cluster` 会被当作 `resourceId` 使用。

## 音色与情绪

音色库在 `web/src/data/voices.json`。

每个音色都包含：

- `id`：传给接口的 `speaker` / `voice_type`。
- `resourceIds`：该音色对应的资源类型。
- `emotions`：官网声明支持的情绪参数；为空时界面会禁用情绪选择。

常用资源：

- `seed-tts-2.0`：语音合成2.0。
- `seed-tts-1.0`：语音合成1.0。
- `seed-tts-1.0-concurr`：语音合成1.0并发版。
- `seed-icl-2.0`：声音复刻2.0。
- `seed-icl-1.0`：声音复刻1.0。

官方参考：

- 语音合成大模型音色列表：https://www.volcengine.com/docs/6561/1257544?lang=zh
- V3 HTTP Chunked 接口：https://www.volcengine.com/docs/6561/1598757?lang=zh

## 功能

- 文本转语音，前端单次最多 5000 字；接口实际限制以火山返回为准。
- 支持火山引擎 V3 HTTP Chunked 接口。
- 支持语速、音调、音量、按音色动态启用情绪强度。
- 支持生成后播放、历史播放、下载、删除、清空。

## 常见错误

- `speaker permission denied`：当前 Resource ID 未授权所选音色。
- `requested resource not granted`：服务实例暂停或未开通。
- `TTSExceededTextLimit`：文本超过接口限制。
